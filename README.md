# solana-subscription

On-chain recurring payments. Subscribers authorize periodic token transfers, merchants collect on schedule. Cancel anytime.

![Rust](https://img.shields.io/badge/Rust-000000?logo=rust&logoColor=white)
![Solana](https://img.shields.io/badge/Solana-9945FF?logo=solana&logoColor=white)
![Anchor](https://img.shields.io/badge/Anchor-blue)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

## Features

- Recurring payment authorization
- Merchant collection
- Subscriber cancellation
- Configurable payment intervals

## Program Instructions

`initialize` | `subscribe` | `collect` | `cancel`

## Build

```bash
anchor build
```

## Test

```bash
anchor test
```

## Deploy

```bash
# Devnet
anchor deploy --provider.cluster devnet

# Mainnet
anchor deploy --provider.cluster mainnet
```
